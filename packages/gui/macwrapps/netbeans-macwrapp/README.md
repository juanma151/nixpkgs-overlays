# netbeans-macwrapp

Mac style app wrapper for netbeans

# nix commands

## build

```bash
nix build github:juanma151/netbeans-macwrapp
```

## install in the user profile

```bash
nix add github:juanma151/netbeans-macwrapp
```

## install to specific profile

```bash
nix add --profile "/path/to/profile" github:juanma151/netbeans-macwrapp
```
